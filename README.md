# Campus Lost & Found Portal

A web-based college campus lost and found system built using HTML, CSS, and JavaScript.

This project helps students and campus administrators report lost items, post found items, search listings, submit claims, and manage item return workflows through a simple and responsive webpage interface.

## Features

- Report lost items
- Report found items
- Search items by name, category, or location
- Filter items by type and status
- Upload or paste item image
- Claim lost/found items
- Admin review panel for claim approval or rejection
- Mark items as returned
- Export records as JSON
- Dark mode support
- Responsive webpage layout

## How to Upload to GitHub

1. Create a new repository on GitHub.
2. Upload all project files (`index.html`, `style.css`, `script.js`, and `README.md`).
3. Commit the files to the `main` branch.

## How to Deploy on GitHub Pages

1. Open your repository on GitHub.
2. Go to **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
